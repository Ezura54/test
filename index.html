Publish

放到 GitHub Pages

当前目录已经是 GitHub Pages 友好的纯静态站点。你只需要把 poppk-knowledge-site 上传到 GitHub 仓库，然后在 Pages 设置中选择分支即可。

推荐部署方式

  1. 新建仓库，例如 poppk-learning-map。
  2. 上传本目录全部文件：index.html、assets/、data/、.nojekyll。
  3. 进入仓库 Settings → Pages。
  4. Source 选择 Deploy from a branch，分支选 main，目录选 /root。
  5. 几分钟后访问 GitHub 给出的 Pages 链接，手机也能直接打开。
