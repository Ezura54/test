Core model

机制导向双通道清除

CL = (WT / 70)^0.75 * (CLnr + CLrenal)
CLrenal = tvCLr * (eGFR / eGFR_median)
V = tvV * (WT / 70)^1.0

教程会反复强调：协变量不是随便加到总清除率上，而是要放在能解释机制的位置。

How to study

建议学习方式

第一次学习按 8 课顺序走；做项目时按案例实操检查；遇到术语再用图谱速查。 每次建模决策都要回答三个问题：机制是否合理、数据是否支持、诊断是否通过。

从这里开始

  1. 先学第 1-4 课，理解 PopPK 基本语言。
  2. 再学第 5-7 课，理解估计、协变量和诊断。
  3. 最后用氨磺必利案例把流程串起来。
  4. 遇到不懂的词，回到图谱速查。
